access graphical environments from everywhere!

 Built for open science

Akhet was built to reproduce complex environments and be able to run simulations without requiring set-up and computing power. We have already prepared images for ROOT, Geant4 and other common environments, all running on distributions used by CERN, Fermilab, and many other top class laboratories.

 Secure

Akhet is secure, providing full encryption at all stages of data transfer. We encrypt connections using SSL and have the ability to run every application on a separate host, isolated from the world. Plus, all of your sensitive data never leaves your datacenter, so that's one thing less to worry about!

 Lightweight

By leveraging on docker technologies, we provide efficient resource utilization. We are also supporting clusters of akhet nodes, so you can scale up your infrastructure without worries!

 Open source

All of our source code is publicly available and released under GNU Affero General Public License. We believe in open licenses!
